### Summary

This app runs on the Laravel framework: https://laravel.com.

### Requirements

1. PHP >= 7.1.3
1. OpenSSL PHP Extension
1. PDO PHP Extension
1. Mbstring PHP Extension

### Local Development

1. php -S localhost:8000 -t public
